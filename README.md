# LLM API

大規模言語モデルを安全に利用するための Web API

## About

`ChatGPT` や `Gemini` など大規模言語モデルに関連するアプリケーションの API を安全に実行するための Web API．
`Secret Manager` などを利用してアクセス制限をかける．

## 実装

`Python` の Web Framework である [FastAPI](https://fastapi.tiangolo.com/ja/) を用いて実装．
